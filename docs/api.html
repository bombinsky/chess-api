/games/{game_id}/moves post

Moves resource

post

Creates move in specific game

post /games/{game_id}/moves

Creates move in specific game

  • Request
  • Response

URI Parameters

  • game_id: required(string)

Body

Media type: application/json

Type: object

Properties
  • move: required(object)
    • from: required(string)
    • to: required(string)

Example:

{
  "move": {
    "from": "A1",
    "to": "A7"
  }
}

HTTP status code 201

Successfull creation

Body

Media type: application/json

Type: object

Properties
  • move: required(object)
    • created_at: required(datetime)
    • from: required(string)
    • to: required(string)
    • captured: (string)
    • special_type: (string)
    • chessman: required(object)
      • type: required(string)
      • color: required(string)

Example:

{
  "move":
  {
    "created_at":"2020-05-09T21:16:33.909Z",
    "from":"A1",
    "to":"A7",
    "captured":"A7",
    "special_type":null,
    "chessman":
      {
        "type":"Rook",
        "color":"white"
      }
  }
}

HTTP status code 404

Game not found

HTTP status code 422

Wrong move - invalid or insufficient params

Body

Media type: application/json

Type: any

Example:

{
  "errors":["incorrect move"]
}
